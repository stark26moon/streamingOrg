Analisis de plataformas de Streaming

Analisis de plataformas de streaming en base al rendimiento de peliculas disponibles en cada plataforma , el analisis se realizo mediante el lenguaje python.Ademas se realizo un analisis, ETL y un analisis EDA hasta llegar a las conclusiones y opciones disponibles para su visualisacion.

Consulta sobre cantidad de peliculas en determinado idioma

Para esta funcion se obtuvo la informacion particular de cada pelicula mediante el analisis paso a paso de la informacion en el diccionario correspondiente a la fila del id de pelicula o titulo buscado los siguientes botones, el primero lleva a la forma /docs de la api para ingresar los parametro de busqueda y el segundo a la respuesta json de la misma api con un parametro cargado,debe ingrear la abreviatura correspondiente al idioma ,de la contenidas en el dataset ,de no ingresar alguna correpondiente, recibira mensaje de ingreso incorrecto e instruccion.

Una vez creada la columna con los nombres de cada película, las consultas se hacen buscando coincidencia exacta con el nombre almacenado.
